Aspect

Internet

World Wide Web (WWW)

Definition

A global network of interconnected networks.
A subset of services and content on the internet, accessed through web browsers.

Infrastructure

Physical routers, switches, data centers, cables.
Web pages, web servers, hyperlinks.

Protocols

Relies on TCP/IP and other communication protocols.
Uses HTTP/HTTPS for data transmission.

Network Types

LANs, WANs, wireless networks, etc.
Interconnected web pages and content.

Connectivity

Facilitates global data exchange and communication.
Allows users to access and interact with web-based content.

Content

Encompasses a wide range of digital services.
Consists of web pages with text, images, videos, and more.
