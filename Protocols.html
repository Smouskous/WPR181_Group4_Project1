Protocols

TCP/IP (Transmission Control Protocol/ Internet Protocol)

This is a set of rules that is made a standard to allow different types of computers to communicated with each other. The IP protocol makes sure that every computer or device that is connected to the internet has its own specific IP serial number called an IP address. TCP says how data should be sent over the internet and how it should be broken down into IP packets

SMTP (Simple Mail Transfer Protocol)

This is for sending and distributing e-mails. This protocol uses the header of the mail to get the id of the receiver and enters the mail into the outgoing queue.

PPP (Point to Point Prototype)

This protocol states the rules using which two devices will authenticate with each other and exchange information with each other.

POP3 stands for Post Office Protocol version 3. It has two Message Access Agents (MAAs) where one is client MAA (Message Access Agent) and another is server MAA(Message Access Agent) for accessing the messages from the mailbox.
